# Prompt.md

## Goal
Build a premium, modern, aesthetically refined personal website for Dhruv Mehta based strictly on the supplied resume content.

## Non-goals
- Do not invent new claims, metrics, certifications, or social proof.
- Do not build a generic portfolio template.
- Do not create a noisy dashboard-like landing page.
- Do not overuse cards or badges in the hero.

## Required deliverables
- A fully working, responsive website
- Strong visual art direction and custom hero visual
- Accurate content based on the provided resume
- Downloadable CV PDF link
- LinkedIn link
- GitHub Pages deployment workflow
- README with local run and deploy instructions
- design/DESIGN.md
- docs/plan.md
- docs/implement.md
- docs/documentation.md

## Visual constraints
- Premium and editorial
- Deep ink / navy background with cyan energy
- Calm, strategic, intelligent tone
- One dominant hero composition
- Two typefaces max
- Sparse, meaningful motion only
- Excellent mobile polish

## Content sections
1. Hero
2. About / profile
3. Experience timeline
4. Capabilities
5. Selected projects
6. Education + tools + contact

## Done when
- site builds successfully
- site works on mobile and desktop
- content is accurate to source
- visual hierarchy is strong
- hero is distinctive and not template-like
- GitHub Pages deploy is configured
- all required docs are present
