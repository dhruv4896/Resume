# Design System Documentation: The Executive Brief

## Overview
Create a light-mode, editorial-formal website for Dhruv Mehta. It should feel professional, aesthetic, formal, and premium. Avoid startup tropes and avoid anything dark, blue-heavy, or cyber-looking.

## Colors
Use warm ivory, pearl, soft stone, and mist as the foundation. Use subtle gradients for atmosphere. The accent family should be quiet and disciplined, leaning taupe, mineral sage, and muted bronze rather than electric blue.

## Typography
Use a serif display font for the main headline and section headings, and a clean sans for body and navigation. Make hierarchy explicit and generous.
- H1 should feel editorial, around 56px to 72px desktop and 36px to 44px mobile.
- Section headings should be around 28px to 36px.
- Body text should sit around 17px to 19px.
- Use generous line height and narrow readable text measures.

## Layout
Keep the site as one page. The hero must read as one composition. No hero cards, no dashboard motifs, no heavy boxed layouts. Use soft gradients, elegant spacing, and refined hierarchy.

## Do's and Don'ts
Do create a light, formal, polished first impression.
Do use gradient fields softly across the hero and section transitions.
Do keep the website calm and executive-adjacent.
Don't use dark surfaces, cyan glows, or generic portfolio card grids.
